AI Context Vault

About

AI Context Vault helps you store and auto-inject context into your AI chat interfaces (ChatGPT, Claude, etc.).

Save snippets of context from anywhere, and they'll be automatically included in your prompts.

Keyboard Shortcuts

  • ⌘I or Ctrl+I: Save highlighted text to your context vault
  • ⌘J or Ctrl+J: Open the context manager overlay
  • Alt+Enter: Prepend context to your message without sending
  • Alt+Shift+Enter: Inject context database and send message immediately

Shortcut Troubleshooting

If the keyboard shortcuts aren't working:

  1. Make sure no other extensions or system shortcuts are using the same key combinations
  2. Try reloading the page or restarting your browser
  3. Check the Chrome extensions page (chrome://extensions/shortcuts) to verify the shortcuts are assigned

Supported Platforms

  • chat.openai.com (ChatGPT)
  • claude.ai
  • *.google.com (Gemini, Bard)
